# Note Swift 🗒️

A modern and minimalist note-taking web app built using React.js and Tailwind CSS.

![Note Swift App](./note-swift-app.png)

## Tech Stack 💻

- Frontend: React.js
- Styling: Tailwind CSS

## Features 🎯

- Add new notes
- Edit existing notes
- Download notes 
- No Login No Signup required
